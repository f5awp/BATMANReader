# BATMAN Watcher

A dispatcher-facing iOS/iPadOS app for American Airlines flight dispatchers.

## Project Structure

- `Sources/`: Application source code organized by domain.
- `Documentation/`: Specifications, architecture maps, and testing guides.
- `BATMANWidgets/`: Widget extension target.
- `Scripts/`: Maintenance and utility scripts.

For detailed documentation, see the [Documentation Index](Documentation/PROJECT_BLUEPRINT.md).
